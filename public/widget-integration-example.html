Chat Widget - Guía de Integración

Este widget de chat permite integrar fácilmente un asistente virtual inteligente en cualquier sitio web. El widget se conecta con tu sistema de gestión de eventos y puede responder preguntas sobre servicios, eventos y contactar con tu negocio.

Método 1: Integración Automática (Recomendado)

Simplemente agrega este script al final del <body> de tu página HTML:

<script src="https://tu-dominio.com/widget.js" data-api-key="TU_API_KEY_AQUI"></script>

Método 2: Integración Programática

Para mayor control, puedes inicializar el widget manualmente:

<script src="https://tu-dominio.com/widget.js"></script> <script> // Inicializar el widget con opciones personalizadas window.ChatWidget.init({ apiKey: 'TU_API_KEY_AQUI', baseUrl: 'https://tu-dominio.com', // Opcional, por defecto usa la URL actual position: 'bottom-right', // 'bottom-right', 'bottom-left', 'top-right', 'top-left' primaryColor: '#3B82F6', // Color principal del widget secondaryColor: '#F3F4F6' // Color de fondo de mensajes del bot }); </script>

Configuración de Parámetros

apiKey (requerido)
Clave API única para autenticar el widget. Obtén esta clave desde tu panel de administración.
baseUrl (opcional)
URL base de tu API. Por defecto usa el dominio actual.
position (opcional)
Posición del widget en la pantalla. Opciones: 'bottom-right' (predeterminado), 'bottom-left', 'top-right', 'top-left'.
primaryColor (opcional)
Color principal del widget (botón, header, mensajes del usuario). Formato hexadecimal.
secondaryColor (opcional)
Color de fondo de los mensajes del bot. Formato hexadecimal.

Características del Widget

  • Interfaz Personalizable: Colores, posición y logo personalizables
  • IA Avanzada: Respuestas inteligentes usando Google Gemini AI
  • Información en Tiempo Real: Acceso a datos actualizados de eventos y servicios
  • Conversaciones Persistentes: Historial de chat mantenido entre sesiones
  • Responsive: Funciona perfectamente en dispositivos móviles y desktop
  • Seguro: Autenticación por API key con límites de tasa

Ejemplos de Personalización

Widget en la esquina inferior izquierda con colores personalizados:

window.ChatWidget.init({ apiKey: 'TU_API_KEY', position: 'bottom-left', primaryColor: '#10B981', // Verde secondaryColor: '#ECFDF5' // Verde claro });

Widget en la esquina superior derecha:

window.ChatWidget.init({ apiKey: 'TU_API_KEY', position: 'top-right', primaryColor: '#8B5CF6', // Púrpura secondaryColor: '#F3F4F6' // Gris claro });

Solución de Problemas

El widget no aparece

  • Verifica que la API key sea correcta y esté activa
  • Asegúrate de que el script se cargue después del contenido de la página
  • Revisa la consola del navegador en busca de errores

El widget no responde

  • Verifica la conexión a internet
  • Comprueba que la URL base sea correcta
  • Revisa si has excedido el límite de tasa de la API

Problemas de estilo

  • El widget usa z-index alto (9999) para aparecer sobre otros elementos
  • Si hay conflictos de CSS, ajusta los colores usando los parámetros
  • El widget es responsive y se adapta automáticamente al tamaño de pantalla

Consideraciones de Seguridad

  • La API key debe mantenerse confidencial
  • No incluyas la API key directamente en el código HTML público
  • Usa variables de entorno o configuración del servidor para la API key
  • El widget incluye límites de tasa para prevenir abuso
